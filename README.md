# Biomedical Imaging Project – SPR-Based Protein 

## Overview
This project presents the design and implementation of a microscopy-integrated Surface Plasmon Resonance (SPR) system for real-time, label-free protein detection.

The system combines optical imaging, signal processing, and quantitative analysis to detect biomolecular interactions through changes in reflected light intensity.

##  Key Contributions
- Designed a full optical imaging system using a 488nm laser, microscope, and camera-based detection
- Implemented real-time data acquisition of image-derived signals
- Analyzed intensity profiles, resonance angle shifts, and FWHM for quantitative measurement
- Processed and visualized imaging data using ImageJ and Python
- Built a prismless SPR system to reduce complexity and improve accessibility

## Methodology Highlights
- Captured real-time reflected light data using a CCD imaging system
- Converted pixel intensity profiles into quantitative measurements
- Tracked resonance angle shifts to detect protein binding events
- Used FWHM analysis to evaluate system sensitivity and accuracy
