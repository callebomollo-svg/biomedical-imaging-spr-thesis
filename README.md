# Biomedical Imaging Project – SPR-Based Protein Detection

## Overview
This project presents a microscopy-integrated Surface Plasmon Resonance (SPR) system for real-time, label-free protein detection.

## Key Contributions
- Designed and implemented an optical imaging system using a laser and microscope
- Captured and analyzed image-based intensity data for biomolecular interactions
- Computed resonance angle shifts and FWHM for quantitative analysis
- Used ImageJ and Python for data processing and visualization

## Relevance to Medical Imaging
This work demonstrates strong experience in:
- Image-based data analysis
- Signal processing from imaging systems
- Quantitative interpretation of biomedical data
